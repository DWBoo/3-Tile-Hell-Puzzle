# 타일 배치 씬</br>
게임에 사용 되는 씬들은 모두 MVC 패턴을 사용.
* SceneController는 로직을 담당
* Model은 로직 처리 후 갱신이 필요한 정보나 데이터를 담는데 사용
* Model의 정보를 가져와서 갱신하여 화면에 표시
